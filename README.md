# TradeStation Backtest Toolkit

Professional toolkit for automated backtesting workflows in TradeStation with Python-based data consolidation and statistical analysis.

## Overview
Tools and utilities for structured, repeatable backtesting research:
- EasyLanguage strategy logging and signal capture
- Configuration grid execution across multi-symbol workspaces
- CSV consolidation and statistical summarization via Python
- Excel report generation with threshold analysis, breadth metrics, and risk profiling

## Tech Stack
- **TradeStation** — EasyLanguage strategy development and chart-based execution
- **Python** — pandas, openpyxl for data processing and reporting
- **Output** — Structured Excel workbooks with multi-tab analysis
